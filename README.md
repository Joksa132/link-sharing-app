# Link-sharing App

## Description

Link-sharing app made as a side project using Next.js 14 with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) inspired by [Frontend Mentor challenge](https://www.frontendmentor.io/challenges/linksharing-app-Fbt7yweGsT).

## Technologies

- [TypeScript](https://www.typescriptlang.org/)
- [ReactJS](https://react.dev/)
- [Next.JS](https://nextjs.org/)
- [Postgresql](https://www.postgresql.org/)
- [Next-Auth](https://next-auth.js.org/)
- [Zod](https://zod.dev/)
- [Tailwind](https://tailwindcss.com/)
- [Uploadthing](https://uploadthing.com/)
- [Notistack](https://notistack.com/)
